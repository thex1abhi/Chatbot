# 🤖 ChatBot: AI Customer Support Bot 

An AI-powered Customer Support Chatbot built for modern websites.  
This platform allows businesses to add an intelligent support assistant to their website in minutes using their own business knowledge.

The chatbot can be embedded into any website and provides real-time automated customer support.


## 🛠 Tech Stack
- **Next.js** — Fullstack React Framework
- **AI model** — Gemini 2.5
- **TypeScript** — TypeSafety
- **Scalekit** — Authentication 
- **MongoDB** — Database
- **React** — UI Library
- **Tailwind CSS** — Styling

## 🚀 Features

- 💬 AI-powered customer support chatbot
- ⚡ Instant responses using business knowledge
- 🔐 Authentication system
- 📊 User Dashboard
- 🌐 Embeddable chatbot script
- 🧠 Knowledge-based responses
- 🎨 Modern clean UI
- 📱 Fully responsive design

🌟 Use Cases
• Ecommerce customer support
•SaaS product helpdesk
•Business FAQ automation
•Website live assistant
•Lead generation chatbot
